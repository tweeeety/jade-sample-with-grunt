# jade-sample-with-grunt
this is jade sample with grunt

## install grunt-cli
```
$ npm install -g grunt-cli
```

## clone jade-sample-with-grunt
```
$ git clone git@github.com:tweeeety/jade-sample-with-grunt.git
```

## initialize package
```
$ npm install
```
